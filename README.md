# OpenKeeb 72

OpenKeeb 72 is an open-source custom mechanical keyboard designed from scratch using KiCad and Onshape.

## Features

- ⌨️ 72-key custom layout
- 🌈 Per-key RGB lighting
- ✨ RGB underglow
- 🔄 Hot-swappable MX switches
- 🎛️ Rotary encoder
- ⚡ RP2040 microcontroller
- 🔌 USB-C
- 🧩 Gasket mount
- ⚪ White PCB
- ⚫ Semi-profile case
- ⚖️ Internal steel weight
- 💻 QMK & Vial support

## Software

- KiCad
- Onshape
- QMK Firmware
- Vial
